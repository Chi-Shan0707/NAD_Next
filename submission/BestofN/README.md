# BestofN Submissions

## Trees
- `extreme8/base/`: canonical Extreme8 exports.
- `extreme8/confidence_sweeps/`: confidence insertion sweeps.
- `extreme12/base/`: stable Extreme12 baselines.
- `extreme12/patches/`: coding-only and LCB-specific patches.
- `extreme12/manifests/`: export metadata.

## Suggested reading order
1. Start from the matching file in `extreme8/base/` or `extreme12/base/`.
2. Compare against a variant in `extreme12/patches/` if the experiment is patch-based.
3. Check `extreme12/manifests/` for provenance when present.
