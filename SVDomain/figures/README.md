# Figures

本目录存放论文图的源文件草稿。

当前使用的是 `mermaid` 源：

- `pipeline_overview.mmd`
- `explainability_stack.mmd`

你可以：

- 直接复制到支持 mermaid 的编辑器
- 导出成 SVG / PNG
- 作为 paper figure 的结构底图

---

## 推荐用途

### `pipeline_overview.mmd`

用于正文方法图，展示：

- extraction positions
- anchor mapping
- domain split
- SVD route
- final selection

### `explainability_stack.mmd`

用于解释性章节，展示：

- model-level summary
- run-level contribution
- decision-level contrast
